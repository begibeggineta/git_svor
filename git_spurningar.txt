1. Hvað gera eftirfarandi Linux skipanir?
	cd - opnar möppu
	cd ~ - opnar möppu í user sem þú ert í C:/User/Bergþór
	ls - sýnir innihald möppuna
	pwd - seigir hvaða möppu þú ert í
	mkdir: Býr til nýa möppu

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone - afritar innihald möppu og situr það í aðra möppu
	git log - seigir hver gerði hvaða kóða 
	git status - sýnir allar gerðir af kóða sem hefur verið gerður
	git diff - sýnir muninn á hvað hefur breyst
	git checkout - gefur það sem er nýtt branch

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff  -sýnir muninn á hvað hefur breyst
	b) git diff --staged - sýnir það sem þú hefur gert
	c) git diff commit1 commit2 - sýnir muninn það sem þú gerðir í möppu1 við möppu2
